Robust Distributed Optimization

Distributed optimization enables a network of agents to solve a global objective using only local data and neighbor-to-neighbor communication. A representative case is federated learning, where each agent retains its own dataset while jointly training a global model that minimizes the aggregate loss.

My work develops robust algorithms that provably converge despite complex agent dynamics, parametric uncertainties, and exogenous disturbances. We co-design the optimizer and controller—leveraging output regulation, extremum seeking, and reinforcement learning—and certify stability and performance using the small-gain theorem and singular perturbation theory. Applications include distributed computation, sensor networks, robotic swarms, and smart grids.

Traffic Signal Optimization

This research focuses on optimizing traffic signal timings to improve urban traffic flow and reduce congestion. The salient features of this work is that modeling and control are purely data-driven, that means, we rely on the data collected from traffic sensors to learn the traffic dynamics and optimize the signal timings. To model the traffic dynamics, we employ machine learning algorithms including linear regreesion, multi-layer perceptor and deep neural networks to capture the complex interactions between traffic signals and vehicle flows. The data is also used to build and calibrate a simulation environment for 42 intersections using SUMO (Simulation of Urban MObility), which allows us to gnerarate synthetic traffic scenarios, test and validate our optimization algorithms in a realistic setting. Moreover, the data is used in a model predictive control (MPC) framework to optimize the traffic signal timings in real-time. We also streamline the data processing and optimization algorithms to ensure that the optimization can be performed within the time constraints of real-world traffic signal systems. These streamline allows us to integrate linear quadratic regulator, gain scheduling, and reinforcement learning algorithm to enhance the performance and robustness of the traffic signal control system.
